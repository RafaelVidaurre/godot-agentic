# godot-agentic
Agentic development framework for godot

## Setup

This is a Godot project set up for developing multiple addons.

- GUT (Godot Unit Test) is added as a submodule in `addons/gut/` for unit testing.

To develop an addon, create a new directory under `addons/` with your addon name, and add the necessary files (e.g., `plugin.cfg`, scripts).

Enable addons in the Godot editor under Project > Project Settings > Plugins.
